## 📁 Project Overview
This project analyzes 31 students' performance across 5 assessments in an English course.

## 📋 Assessments
- Quiz 1
- Quiz 2  
- Quiz 3 (Online)
- Midterm
- Final Exam

## 🎯 Key Questions Answered
1. How did the class perform overall?
2. Which assessment was easiest/hardest?
3. Did students improve throughout the course?
4. What factors predict final exam performance?
5. Who are the top/bottom performers?

## 📈 Key Findings
- **Class Average:** 12.1/20 (60.5%) — below expectations
- **Most Challenging:** Midterm (average [15]/20)
- **Easiest:** first quiz (average [17.4]/20)
- **Correlation:** Strong correlation (r=0.93) between Midterm and Final
- **Top Performer:** [Elham ] ([14.7]/20)


## 📊 Visualizations
The notebook includes the following visualizations:
